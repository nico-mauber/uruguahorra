# Assets - Uruguahorra App

## Iconos necesarios:

- **icon.png**: 1024x1024px - Icono principal de la app
- **adaptive-icon.png**: 1024x1024px - Icono adaptativo para Android
- **splash.png**: 1284x2778px - Pantalla de carga
- **favicon.png**: 32x32px - Favicon para web

## Generación de iconos

### Opción 1: Usar herramientas online
1. Visita https://www.canva.com o https://figma.com
2. Crea un diseño de 1024x1024px con fondo #6366F1
3. Exporta como PNG

### Opción 2: Usar el generador de Expo (recomendado)
```bash
npx create-expo-app --template
```

### Colores de marca:
- Principal: #6366F1 (Indigo)
- Secundario: #10B981 (Verde)
- Fondo: #FFFFFF

## Iconos temporales

Por ahora, la app puede funcionar sin los iconos. Expo usará iconos por defecto.
